The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

# 📚 The Vault Bookstore Index

An ultra-snappy, high-contrast **Neo-Brutalist bookstore dashboard** engineered to showcase curated literary archives, dynamic access rules, and fluid interactive tabs. 

This layout rejects traditional modern web trends—throwing out heavy image asset dependencies, hidden nested sub-menus, and slow decorative loader animations—in favor of raw, lightning-fast structural integrity.

---

## 🛠️ Design & Interface Features

* **Interactive Folder-Tab Interface:** Features fluid, physics-based custom animations when switching views via `framer-motion`.
* **High-Contrast Neo-Brutalism:** Styled with thick, intentional dark borders (`border-4 border-[#222]`), vintage flat offset dropshadows (`shadow-[12px_12px_0px_0px_#222]`), and vivid mint/yellow accent points.
* **Infinite Text Ticker Strips:** A continuous looping marquee tracking live archival data across client viewports.
* **Archival Integrity Metrics:** An interface blueprint mapping out premium vault collections, access restrictions, and local inventory models cleanly.

---

## 📦 Requirements & Tech Stack

This application runs purely inside client browsers and comes pre-configured with all packages handled. The architecture includes:

* **Vite + React 18 Engine** (Core SPA virtual DOM layer)
* **Tailwind CSS Utility Framework** (Layout structural constraints)
* **Framer Motion Engine** (GPU-driven interface animations)
* **React Icons Module** (Interface indicator glyphs)
